# 📊 Angular Dashboard - E-Commerce Admin Panel

[![Angular](https://img.shields.io/badge/Angular-17-blue)](https://angular.io/)
[![Bootstrap](https://img.shields.io/badge/Bootstrap-5.3-blue)](https://getbootstrap.com/)

## 🎯 Features

- **Product Management** - Inventory tracking and product catalogs
- **Order Tracking** - Real-time order status monitoring
- **Customer Analytics** - Sales data visualization
- **Admin Panel** - Role-based access control

## 🛠️ Development Setup

```bash
npm install
ng serve
```

## 🚀 Deployment

### Deploy to GitHub Pages

1. Create `angular.json` production configuration
2. Build: `npm run build`
3. Upload `dist/` folder to GitHub
4. Enable GitHub Pages in repository settings

### Deploy to Netlify/Vercel

```bash
npm run build
# Deploy the dist/ folder
```
