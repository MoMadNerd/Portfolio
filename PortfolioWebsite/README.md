## 🛠️ Development Setup

### Prerequisites

- Node.js 18+ installed
- Angular CLI 17+ installed

### Installation

```bash
# Install dependencies
npm install

# Start development server
ng serve

# Open browser
http://localhost:4200
```

### Build for Production

```bash
npm run build
ng build --configuration=production
```

The production build will be in `dist/portfolio/browser/`.
