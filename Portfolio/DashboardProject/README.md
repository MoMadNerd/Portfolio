# 📊 Data Visualization Dashboard

An Angular dashboard for displaying analytics, statistics, and data visualizations.

## Tech Stack

- **Framework:** Angular 17+
- **UI:** Bootstrap 5, Chart.js
- **Purpose:** Frontend-only demo project (connects to mock API)

## Features

- 💫 Real-time data visualization
- 📈 Revenue charts and analytics
- 👥 User activity tracking
- 📋 Task board with Kanban view
- 🌤️ Weather widget
- 📊 Login activity logs

## Getting Started

```bash
npm install
ng serve
```

Visit `http://localhost:4200` to see the dashboard.
